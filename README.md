London Explorer
Project Description
London Explorer is a tourism website dedicated to the city of London. The website provides information about famous attractions, travel tips, useful articles, and general information for tourists who want to visit the capital of the United Kingdom.

The project was developed using React for the frontend, Tailwind CSS for styling, and Strapi as a Headless CMS for managing content on the About page.

Website Pages
Home
The main page of the website, containing a hero section and information about London's most important attractions and travel opportunities.

About
A page connected to Strapi CMS that presents information about the website and its purpose.

Attractions
A page containing information about the most popular tourist attractions in London, including Big Ben, Tower Bridge, London Eye, Buckingham Palace, British Museum, and Hyde Park.

Articles
A page with travel-related articles and recommendations for visitors.

Contact
A contact page containing a simple contact form.
